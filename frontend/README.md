# CodeGenesis Frontend

CodeGenesis is an AI-powered code generation platform that allows users to generate full-stack applications without writing a single line of code.

This directory contains the Vue 3 + TypeScript frontend application.

## Project Setup

### Install Dependencies

```sh
npm install
```

### Development Server

```sh
npm run dev
```

### Build for Production

```sh
npm run build
```

### Linting

```sh
npm run lint
```
