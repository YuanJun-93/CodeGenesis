# CodeGenesis

Backend for AI Native No-Code Application Generation Platform.

## Quick Start

### Prerequisites
- Go 1.20+
- Make

### Usage

**1. Start server**
```bash
make run
```
*Server listens on `0.0.0.0:8888` by default.*

**2. Health Check**
```bash
curl http://localhost:8888/api/healthz
```
